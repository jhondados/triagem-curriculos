# 👥 Triagem Inteligente de Currículos

> Triagem de currículos com IA e matching semântico. Rankeia candidatos por fit com a vaga, detecta viés e gera perguntas de entrevista.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/triagem-curriculos/

[![Vertical](https://img.shields.io/badge/Vertical-RH-blue)]() [![IA](https://img.shields.io/badge/IA-NLP-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**94%** Match · **10K/dia** Currículos · **-85%** Tempo · **monitorado** Viés

## 🧠 Tecnologias de IA
NLP · LLM · Semantic Matching

## ✨ O que faz
- Matching semântico vaga x perfil
- Ranking de candidatos
- Detecção de viés
- Perguntas de entrevista geradas

## 🗄️ Fonte de dados real
[CAGED + currículos](https://www.gov.br/trabalho)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com CAGED + currículos
2. **Processamento com IA** — NLP · LLM · Semantic Matching
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
